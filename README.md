# Odin Rock Paper Scissors

A simple **Rock Paper Scissors** game, created as part of **The Odin Project (TOP)** exercises.  
This project is focused on practicing **JavaScript basics**, including functions, conditionals, and basic game logic.

## About the Project

This is a console-based Rock Paper Scissors game where the player competes against the computer.

The game includes:
- Random computer choices
- Player input using prompt
- Case-insensitive input handling
- Score tracking for both players
- A final winner announcement after 5 rounds

Each round prints the result directly to the browser console.

## How to Play

1. Open the project in your browser
2. Open the **Developer Console**
3. You will be prompted to enter **Rock**, **Paper**, or **Scissors**
4. The game plays **5 rounds** and displays the final score
